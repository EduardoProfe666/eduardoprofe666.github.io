<p>El paquete es una migración de todas las funcionalidades
desarrolladas en el proyecto
de <a href="https://github.com/EduardoProfe666/Matematica-Numerica-Google-Colab/">Matemática Numérica</a>.</p>
<p>Contiene implementaciones de los principales métodos numéricos dados
en Matemática Numérica.</p>
